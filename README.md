# Inio server
Backend server that tracks crypto portfolio across different exchanges.

## Env variables
Should setup in `.env`:
- POSTGRES_HOST
- POSTGRES_PORT
- POSTGRES_USERNAME
- POSTGRES_PASSWORD
- POSTGRES_DB
- TS_NODE_DEV=true
- DOMAIN=localhost
- PORT=5000 
- CURRENCY_KEY - api key from nocodeapi.com Currency Exchange
- ETHERSCAN_API_KEY
- BSCSCAN_API_KEY
